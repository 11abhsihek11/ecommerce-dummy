# ecommerce-dummy

Small demo static site for a mock ecommerce store. The full site lives in `index.html` (static HTML/JS/CSS).

Author: aman abhi

How to view locally:

1. Open `index.html` in your browser (double-click the file or use `Start-Process index.html` in PowerShell).

2. To publish publicly via GitHub Pages:
   - Go to: https://github.com/11abhsihek11/ecommerce-dummy
   - Settings → Pages → Source: Branch `main`, Folder `/ (root)` → Save.
   - GitHub Pages URL will usually be: `https://11abhsihek11.github.io/ecommerce-dummy/` (it may take a minute to publish).
